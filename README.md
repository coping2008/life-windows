🚀 人生重构计划 (Life Window Monitor) V8.0
“人生不是一场长跑，而是一系列有限且不可逆的窗口期。”

这是一个基于 React + Vite 构建的高性能可视化工具，旨在帮助用户识别、监测并把握人生中关于学业、职场、财富、健康及人际关系的 46 个核心关键窗口。

🌟 核心特性
📊 全量系统监测：内置 5 大维度（职场、财富、健康、认知、关系）、46 条深度调研的人生窗口数据。

🎛️ 实时交互体验：通过动态年龄滑动块，实时观察窗口的开启、关闭及未来趋势。

⚡ 视觉化预警：

🔓 开放中：绿色高亮，提示当前最佳行动期。

🔒 已锁死：红色置灰，警示错过窗口后的高昂补救代价。

🕒 未来时：蓝色提示，辅助进行前瞻性规划。

🛠️ 自定义扩展：支持添加个人专属的人生窗口，数据通过 LocalStorage 实现本地持久化。

📱 响应式设计：提供“网格看板”与“紧凑条目”双模式切换，适配各种屏幕尺寸。

✨ 灵动交互：基于 Motion (Framer Motion) 实现丝滑的物理弹性动画与转场特效。

🛠️ 技术栈
前端框架: React 19

构建工具: Vite 6

样式处理: Tailwind CSS 4

动画引擎: Motion (原 Framer Motion)

图标库: Lucide React

类型系统: TypeScript

📖 数据维度说明
本工具监测的窗口包括但不限于：

学业与职场：第一学历跃迁、应届生身份、职业背书红利等。

财富与资产：财务复利启动、大宗资产加杠杆、保险配置底线等。

身体健康：骨密度储蓄期、基础代谢重构、重大疾病早筛等。

认知心智：语言学习黄金期、心智原动力保留、性格分化定格等。

人际关系：婚恋高配匹配、亲子早期陪伴、老人安宁照护等。
